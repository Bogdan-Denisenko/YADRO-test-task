## Запуск

Для сборки Docker-образа  необходимо из корневой дирректории проекта выполнить следующую команду:  
`docker build -t {ИМЯ_ОБРАЗА} .`

После того, как образ собран, можно запустить образ в докер контейнере при помощи команды:  
`docker run -e FILE_PATH={ПУТЬ_ДО_ФАЙЛА_С_ВХОДНЫМИ_ДАННЫМИ} --name {ИМЯ_КОНТЕЙНЕРА} {ИМЯ_ОБРАЗА}`

Для повторного запуска контейнера необходимо использовать команду:  
`docker start -a {ИМЯ_КОНТЕЙНЕРА}`

### Пример запуска

Сборка:  
`docker build -t computer_club .`

Запуск контейнера:  
`docker run -e FILE_PATH=testData/input1.txt --name computer_club_container computer_club`

Перезапуск:  
`docker start -a computer_club_container`
